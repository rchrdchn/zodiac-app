Zodiac JavaScript Application

Project Overview

I built this Zodiac Application using mainly JavaScript to manipulate the DOM. I've always liked reading the horoscope and often found it fun and funny at the same time because it was either a hit or miss thing - rarely accurate, but always entertaining.

Process

I contrusted an array with several objects with keys and values, then created a function with a for loop, a few variables, and a if/else statement that controled the outcome of the user input when the "Click Me!" button was clicked.

Technology Used

JavaScript, jQuery, HTML5, CSS3, CSS Animation
